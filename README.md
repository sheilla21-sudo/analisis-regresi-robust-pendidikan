# Analisis Regresi Robust Pendidikan Indonesia 2024

Repositori ini berisi data dan script analisis regresi robust untuk mengkaji pengaruh faktor-faktor pendidikan terhadap persentase peserta didik yang bekerja di Indonesia tahun 2024.

## Data
Data bersumber dari Badan Pusat Statistik (BPS) 2024.

## Metode
Metode yang digunakan adalah regresi robust dengan pendekatan M-estimator menggunakan R.

## Isi Repository
- Data: file Excel penelitian
- Script: kode R untuk analisis
